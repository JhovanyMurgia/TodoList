# Metodologia

O projeto foi organizado em etapas para aproximar a implementacao de um fluxo profissional de desenvolvimento.

## Etapas

1. Levantamento dos requisitos funcionais.
2. Definicao da arquitetura MVC.
3. Especificacao do contrato da API.
4. Implementacao do model, controller e rotas.
5. Desenvolvimento da interface.
6. Criacao dos testes automatizados.
7. Documentacao e deploy.

## Decisoes

- O armazenamento e feito em memoria para manter o escopo simples.
- A interface usa JavaScript puro, sem frameworks.
- A API usa JSON como formato padrao.
- Os testes cobrem regras do model e comportamento dos endpoints.
